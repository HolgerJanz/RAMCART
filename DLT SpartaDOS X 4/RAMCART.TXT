RAMCART - Ramdisk for The!Cart

RAMCART [driveNumber|driveLetter]

RAMCART is a ramdisk driver for
THE!CART. driveNumber specifies the
drive number that should be used for
the ramdisk or driveLetter specifies
the drive letter that should be used
for the ramdisk, e.g. 8 or H. If no
drive is specified then the maximum
drive defined by the environment
variable MAXDRV is used.

Ramdisk will be formatted with 4096
sectors of 128 Bytes and SpartaDOS
file system.

Do not forget to use SHIFT with
RETURN / Fire in the main menu of
The!Cart to enable the access to the
RAM and configuration of The!Cart.
